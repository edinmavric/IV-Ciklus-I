# Sistem za zakazivanje termina i red čekanja

Sistem za zakazivanje termina i red čekanja za frizere, klinike, servise i druge usluge.

## Stack

- **Next.js** (App Router)
- **NextAuth** (Credentials + OAuth)
- **MongoDB**
- **Server Actions**

## Funkcionalnosti

- **Role**: Admin / Worker / User
- **Zakazivanje termina** - jednostavna rezervacija
- **Live queue** - pregled redosleda čekanja
- **Otkazivanje / pomeranje termina** - fleksibilna promena
- **Admin dashboard** - grafici i statistika
- **Email notifikacije** - obaveštenja za korisnike
